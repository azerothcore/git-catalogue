How to add an item in the catalogue

In order to add your repository in the catalogue, you need to add the github topic related to each tab.

Tabs & topics

  • {{ tab.key }}: {{ tab.value }}

Add these information in your project README

  • Name:   name of your script.
  • Author:   username of the original author (in case it's forked into an organization).
  • Repository:   link to the GitHub repository.
  • Download:   link to download the sources as .zip (take it from GitHub).
  • License:   license used to release the code.

Description

Give a complete description of what does the script and how it does it.

Optional / Recommended

  • Add a README.md in the root folder with a complete description of your script. It will be shown in the details page of the catalogue.
  • Add an icon.png in the root folder. It will be shown in the script list and details page of the catalogue.
  • Use the db_assembler to include your SQL patches.
  • Follow best practices from the skeleton (.gitattributes, .gitkeep etc...)
  • Give the user the biggest amount of options for configuration possible. It reduces the need to modify the code because "something doesn't work as I would like to".
  • Use the GNU Affero General Public License v3.0.
